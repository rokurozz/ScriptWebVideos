# Script - Botões para videos na Web (Feito para Mobile) - versão para pc no final

Um script simples que adiciona algumas funcionalidades extras para vídeos na web, com salvamento automático de progresso, útil para pessoas que desejam assistir animes, séries e filmes através de sites

## ✨ Funcionalidades Principais

### 🎯 Controles 
- **Menu flutuante** com botões de acesso rápido

### 💾 Salvamento Automático
- **Progresso salvo automaticamente** a cada 15 segundos assistidos, ou ao sair, podendo também alterar o tempo de salvamento entrando no menu através do botão "⚙️" no controlsbar
- **Botão "Continuar de onde parou"** com swipe para remover (Basta apenas deslizar para o lado caso queira remover)

### ⭐ Botões Disponíveis:
| **1x** | Velocidade: Alterna entre 1x, 1.25x, 1.5x, 2x

| **⤺10** | Voltar 10s

| **⤻10** | Avançar 10s

| **⛶** | Modo tela cheia

| **🔓** | Bloquear toques na tela | Bloqueia/desbloqueia

| **🪄** | Faz pequenos ajustes nas cores do video 

| **⚙️** | Menu | Gerencia algumas funções do script e ativa outras por escolha do usuário

### 🔒 Sistema de Bloqueio
- **Bloqueio de gestos** para evitar toques acidentais
- **Botão de desbloqueio rápido** aparece quando gestos estão bloqueados

### 📋 Blacklist Integrada
- **Sites pré-bloqueados:** YouTube, Instagram, Facebook, X, TikTok, Netflix, Bilibili, Crunchyroll
- **Interface amigável** para gerenciar a blacklist, através do botão menu "⚙️" 

## 🚀 Instalação

### Pré-requisitos
- Navegador para Android com suporte a extensões (Firefox, Edge, e outros)
- Recomendo o Edge ou Firefox 
- Extensão [Tampermonkey](https://www.tampermonkey.net/) instalada

### Passo a Passo

1. **instale a extensão do tempermonkey**, digite tempermonkey extension na barra de busca, e instale em seu navegador
2. **Abra o Tampermonkey** no seu navegador
3. **Vá para a aba "Utilitários"** (Utilities)
4. **Cole o link abaixo** no campo "Importar URL" (Install from URL):
https://raw.githubusercontent.com/rokurozz/ScriptWebVideos/main/script.user.js
5. **Clique em "Instalar"** (Install)
6. **Confirme a instalação** clicando em "Instalar" novamente
